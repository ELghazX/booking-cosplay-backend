# Penyedia API Booking Cosplay ChocoMintCos

# Cara menjalankan

## Dengan Docker

```shell
docker compose up -d
```

Kemudian

```shell
mvn spring-boot:run
```

## Dengan xampp

Tinggal nyalain aja, sesuaikan port dengan application properties
kemudian jalankan dengan

```shell
mvn spring-boot:run
```
